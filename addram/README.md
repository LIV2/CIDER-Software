# Addram

A tool to add the BonusRAM ($A0000) to the system

## Usage
```
    -d            - Perform a dry-run
    -v            - Verbose
    -m            - Attempt to merge Bonus RAM with Fast RAM Memory block
    -p <priority> - Set Fast / BonusRAM Priority
```
